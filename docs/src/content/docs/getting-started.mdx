title: Getting Started
---

Add a LLM Button to your Starlight or Astro project allowing anyone to copy the markdown of any page to paste into a LLM.

## Prerequisites

You will need to have a Starlight website set up.
If you don't have one yet, you can follow the ["Getting Started"](https://starlight.astro.build/getting-started) guide in the Starlight docs to create one.

## Installation
Starlight [configuration](https://starlight.astro.build/reference/configuration/#plugins) in the `astro.config.mjs` file.

   ```diff lang="js"
   // astro.config.mjs
   import starlight from '@astrojs/starlight'
   import { defineConfig } from 'astro/config'
   +import starlightLLMButton from 'starlight-llm-button'

   export default defineConfig({
     integrations: [
       starlight({
   +      plugins: [starlightLLMButton()],
         title: 'My Docs',
       }),
     ],
   })
   ```

3. [Start the development server](https://starlight.astro.build/getting-started/#start-the-development-server) to preview the plugin in action.

The Starlight LLM Button plugin behavior can be tweaked using various [configuration options](/configuration).

## Component overrides

The Starlight LLM Button plugin uses a Starlight [component override](https://starlight.astro.build/guides/overriding-components/) for the [`TableOfContents`](https://starlight.astro.build/reference/overrides/#tableofcontents) component to add the `LLMButton` component to the top of the Table of Contents.

If you have a custom `TableOfContents` component override in your Starlight project, you will need to manually render the `TableOfContents` component from the Starlight LLM Button plugin in your custom component:

```diff lang="astro"
---
// src/components/overrides/TableOfContents.astro
import Default from '@astrojs/starlight/components/TableOfContents.astro'
+import StarlightLLMButtonTableOfContents from 'starlight-llm-button/components/TableOfContents.astro'
---

<Default><slot /></Default>
<p>Custom content in the TableOfContents override</p>
+<StarlightLLMButtonTableOfContents />
```
